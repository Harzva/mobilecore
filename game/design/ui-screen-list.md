# TuiMa Push UI Screen List

## 1. Brand Visual Direction

视觉关键词：

```text
TuiMa 推嘛
推箱子，但推的是大模型
简约、可爱、清新、浅色、柔和渐变
Mint / Teal / Sky Blue / Lavender
圆角卡片、软 3D 模型箱子、可爱 mascot
```

核心形象：

```text
小推嘛角色：绿色/薄荷色，有小芽，负责推动模型箱子
模型箱子：发光立方体，带 AI / neural nodes 图案
手机目标格：带绿色勾的 phone tile
```

## 2. Screen 01: Game Home

页面目标：让用户理解这不是普通跑分，而是游戏化 benchmark。

核心内容：

```text
Header: TuiMa 推嘛
Score badge: 12,560
Hero: Push models on your phone
Sub: Play, push, and discover which models your phone can run.
Cards: Start Challenge / Custom Grid / Leaderboard
Progress Path: 0.5B → 1.5B → 3B → 7B → 14B → Phone ✓
Stats: Best model / Best score / Avg speed
```

## 3. Screen 02: Gameplay Board

页面目标：展示实际推箱子玩法。

核心内容：

```text
Level: Device Challenge 04
Score / Moves / Speed Bonus
8x8 Board
Model Boxes: 0.5B, 1.5B, 3B, 7B, 14B
Target Phone Tiles
Walls / Blocks
Mascot player
Undo / Reset / Hint
Live benchmark stats
Footer message
```

视觉重点：

```text
不同模型箱子从小到大、从绿到蓝到紫
推到目标格后目标手机出现绿色勾
通关后出现星星徽章
```

页面目标：支持用户自定义格子和分享自己的 benchmark puzzle。

核心内容：

```text
Device Preset: Smartphone / Tablet / Custom
Score Rule: Sum of Values / Highest Tile
Tile Palette
```

```text
Speed Boost
Upload Badge
```

## 5. Screen 04: Result Upload

页面目标：告诉用户成绩完成，可以上传排行榜。

核心内容：

```text
Challenge Complete
Total Score
Avg Speed
Best Cleared Model
Stages Completed
Upload Result

```

```text
Only device class, score, average speed, and cleared models are uploaded.
No personal data.
```
